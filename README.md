# 从源码编译
1. 安装 Rust 环境
2. 安装 Nodejs 环境
3. 开始编译
   ```shell
   yanr tauri build
   # OR
   npm run tauri build
   ```
4. 编译完成后，可在 `./src-tauri/target/release` 目录下找到可执行文件
5. 安装包在 `./src-tauri/target/release/bundle` 目录下
